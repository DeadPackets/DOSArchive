Written also in 1993, this text editor was Yasser's interview assignment to join his first company for his first official professional job. The assignment was to write a text editor that was capable of saving/loading, scrolling, having a toggle-able insertion mode, and supporting cursor movement.

Written in 1993.
Language: C

Usage:

EDIT.EXE - The original editor writtin in Yasser's own inspired GUI.
EDITOR.EXE - The same editor in an alternate GUI inspired by Windows.

Controls:

F10 - Access the top menus.
INS - Toggle text insertion mode.
